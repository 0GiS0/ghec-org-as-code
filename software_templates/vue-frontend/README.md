# Vue.js Frontend Template

This template scaffolds a modern Vue.js frontend application with TypeScript, Vite, and best practices.

## Features

- **Vue 3**: Latest version of Vue.js with Composition API
- **TypeScript**: Optional TypeScript support for type safety
- **Vite**: Fast build tool and development server
- **Pinia**: Optional state management (Vue's official state management)
- **Vue Router**: Optional routing for single-page applications
- **Tailwind CSS**: Optional utility-first CSS framework
- **DevContainer Ready**: Pre-configured development environment

## Usage

1. Fill in the project name and description.
2. Select the owning team.
3. Choose your preferred configuration options.
4. The template will generate a repository with the necessary structure.

## Included Tools in DevContainer

- Node.js LTS
- npm / pnpm
- Vue CLI
- Vite
- ESLint + Prettier
