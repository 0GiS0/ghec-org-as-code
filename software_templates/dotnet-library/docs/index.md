# .NET Library Template

This Backstage template helps you create reusable .NET class libraries.

## Overview

Create professional .NET libraries ready for sharing across projects and publishing to NuGet.

## Features

- **Modern .NET**: Support for .NET 8.0, 9.0, and .NET Standard 2.1
- **NuGet Ready**: Pre-configured for package publishing
- **Full Testing**: xUnit with FluentAssertions
- **CI/CD**: GitHub Actions pipeline included
- **Documentation**: TechDocs integration

## Getting Started

1. Use this template from the Backstage catalog
2. Fill in the required parameters
3. Clone the generated repository
4. Start coding!
